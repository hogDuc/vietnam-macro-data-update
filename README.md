### Create Google API credentials for Notification Email

1. Go to https://console.cloud.google.com/apis/credentials 
2. Create OAuth 2.0 client ID -> Application Type: Desktop App, save the credential json file as "credentials.json" in "creds" folder.

### Download Chrome driver
1. Download [Chrome binary for Windows ](https://storage.googleapis.com/chrome-for-testing-public/142.0.7444.175/win64/chrome-win64.zip). Save as folder 'chrome-win64' in the source folder
2. Download [Chrome driver for Windows](https://storage.googleapis.com/chrome-for-testing-public/142.0.7444.175/win64/chromedriver-win64.zip). Save as folder 'chromedriver-win64' in the source folder

### Environment variables
.env file should be saved in the source folder
